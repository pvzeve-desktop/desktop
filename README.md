# 宇宙大战桌面版

[![License](https://img.shields.io/github/license/[你的用户名]/CosmicWarDesktop)](https://github.com/[你的用户名]/CosmicWarDesktop/blob/main/LICENSE)

一款专为体验《宇宙大战》网页游戏打造的桌面应用，内置Flash环境支持，可流畅访问并浏览pvzbaike.com网站，让你在桌面端沉浸式体验经典的宇宙大战网页游戏，适配Windows 10/11系统。

## 📫 核心功能
- 内置Flash运行环境，无需额外安装插件即可访问pvzbaike.com
- 专向适配《宇宙大战》网页游戏，保障游戏流畅运行
- 支持桌面端全屏/窗口模式游玩，优化视觉体验
- 支持启动后自动加载pvzbaike.com宇宙大战游戏页面

### 使用说明补充
1. 下载并安装应用后，直接启动即可自动加载pvzbaike.com的宇宙大战游戏页面；
2. 应用已内置Flash兼容环境，无需手动配置，开箱即用。

### 注意事项
- 运行时请保证网络通畅，以正常加载游戏资源；
- 目前仅支持Windows 10/11 64位系统，暂不支持其他操作系统；
- 若遇到游戏加载异常，可尝试重启应用或检查网络连接。
